HTML & CSS

HTML e CSS sono il cuore dello sviluppo web. L'HTML definisce come è strutturata una pagina web e i CSS ne definiscono lo stile. Ho una conoscenza fondamentale di html5 e css3 per la creazione di layout reattivi user friendly.

JAVASCRIPT

JavaScript è il linguaggio di programmazione più popolare per Frontend, JavaScript aiuta gli sviluppatori web a rendere le pagine web dinamiche e interattive. Ho una conoscenza base di javascript per effettuare piccole interazioni con l'utente.

Frameworks

I framework frontend semplificano il lavoro di sviluppatori e designer web e offrono un'ottima esperienza di sviluppo. Ho una conoscenza di base dei framework frontend come Angular, Bootstrap, jquery e React

JAVA

Java è un linguaggio di programmazione ad alto livello, orientato agli oggetti, che consente di sviluppare applicazioni sicure e robuste su qualsiasi dispositivo dotato di JVM. Ho raggiunto un livello intermedio di competenza in Java, comprendendo e applicando concetti come l'ereditarietà, l'incapsulamento, il polimorfismo, lavorando con collezioni, gestendo eccezioni e utilizzando API standard.

SPRING

Spring è un framework Java che facilita lo sviluppo di applicazioni robuste e scalabili, riducendo la complessità attraverso la gestione automatica delle dipendenze e l'inversione del controllo. Con un livello intermedio di competenza in Spring, sono in grado di configurare e utilizzare componenti fondamentali come Spring Boot, Spring MVC e Spring Data, per creare applicazioni web efficienti e integrate con database.

.NET

.NET è un framework di sviluppo software creato da Microsoft, che supporta la costruzione e l'esecuzione di applicazioni su Windows, macOS e Linux. Con un livello intermedio di competenza in .NET, sono in grado di sviluppare applicazioni web, desktop e servizi utilizzando C# e ASP.NET. Questo include la gestione di entità di database con Entity Framework, la creazione di API RESTful e l'implementazione di architetture MVC.

SQL

SQL (Structured Query Language) è un linguaggio standard per la gestione e la manipolazione di database relazionali. Con un livello intermedio di competenza in SQL, sono in grado di scrivere query complesse per estrarre, inserire, aggiornare e cancellare dati, creare e gestire tabelle, indici e viste.

MiCROSERVIZI

I microservizi sono un'architettura software che suddivide le applicazioni in servizi piccoli, indipendenti e interconnessi, ognuno eseguendo una singola funzione. Con un livello intermedio di competenza nei microservizi, sono in grado di progettare, sviluppare e gestire sistemi distribuiti utilizzando tecnologie come Docker, Kubernetes e API RESTful.
